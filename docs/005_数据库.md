# 数据库设计与使用指南

## 数据库概述

LOF 基金套利监控系统使用 SQLite 数据库存储数据，替代了之前的 CSV 文件存储方式。数据库文件位于 `arbcore/database/arb_data.db`。

## 数据库表结构

### 汇率表 (exchange_rates)

| 字段名 | 数据类型 | 描述 |
|-------|---------|------|
| date | TEXT | 日期 (YYYY-MM-DD) |
| rate_type | TEXT | 汇率类型 (如: CNY_USD) |
| rate | REAL | 汇率值 |
| source | TEXT | 数据来源 |
| timestamp | TEXT | 数据获取时间 |

### ETF 数据表 (etf_data)


### LOF 基金数据表 (lof_data)

| 字段名 | 数据类型 | 描述 |
|-------|---------|------|
| date | TEXT | 日期 (YYYY-MM-DD) |
| code | TEXT | 基金代码 |
| nav | REAL | 净值 |
| price | REAL | 收盘价 |
| static_valuation | REAL | 静态官方估值 |
| futures_valuation | REAL | 期货校准估值 |
| native_futures_valuation | REAL | 期货原生估值 |
| premium_rate | REAL | 溢价率 |
| hedging_value | REAL | 对冲值 |
| calibration_value | REAL | 校准值 |
| source | TEXT | 数据来源 |
| timestamp | TEXT | 数据获取时间 |


### 校准值 (Calibration Value)

校准值是通过LOF净值与 ETF 价格的关系计算得出，存储在 `lof_data` 表的 `calibration_value` 字段中。


### 对冲值 (Hedging Value)

对冲值是用于对冲市场风险的参考值，存储在 `lof_data` 表的 `hedging_value` 字段中。

### 期货结算价表 (futures_settlement)

| 字段名 | 数据类型 | 描述 |
|-------|---------|------|
| date | TEXT | 日期 (YYYY-MM-DD) |
| symbol | TEXT | 期货合约代码 (如: GC, CL, NQ, ES) |
| settlement_price | REAL | 结算价 |
| source | TEXT | 数据来源 |
| timestamp | TEXT | 数据获取时间 |

## 数据库管理

### 数据库初始化

数据库在首次运行时自动初始化，创建所需的表结构。初始化过程由 `db_manager.py` 中的 `initialize_database` 方法完成。

### 数据导入

- **API 数据导入**：通过 `data_fetcher.py` 从 API 获取数据并导入数据库

### 数据查询

系统提供了多种数据查询方法，主要通过 `db_manager.py` 中的方法实现：

- `get_exchange_rate`：获取指定日期的汇率
- `get_futures_settlement`：获取指定日期的期货结算价
- `get_lof_data`：获取指定基金的历史数据
- `get_latest_lof_data`：获取基金的最新数据
- `get_system_health`：获取系统健康状态

## 数据库使用指南

### 1. 直接查询数据库

可以使用 SQLite 命令行工具或 GUI 工具（如 DB Browser for SQLite）直接查询数据库：

```bash
# 使用 SQLite 命令行工具
sqlite3 arbcore/database/arb_data.db

# 查询最新的汇率数据
SELECT * FROM exchange_rates ORDER BY date DESC LIMIT 10;

# 查询特定基金的历史数据
SELECT * FROM lof_data WHERE code = '501018' ORDER BY date DESC LIMIT 30;
```

### 2. 通过代码查询数据库

使用系统提供的 `DatabaseManager` 类查询数据库：

```python
from arbcore.database.db_manager import DatabaseManager

# 初始化数据库管理器
db = DatabaseManager()

# 获取最新汇率
rate = db.get_exchange_rate('2023-07-01', 'CNY_USD')

# 获取基金最新数据
data = db.get_latest_lof_data('501018')

# 获取系统健康状态
health = db.get_system_health()
```

### 3. 数据备份与恢复

**备份数据库**：

```bash
# 复制数据库文件
copy arbcore\database\arb_data.db arbcore\database\arb_data_backup.db
```

**恢复数据库**：

```bash
# 替换数据库文件
copy arbcore\database\arb_data_backup.db arbcore\database\arb_data.db
```
